PN2 RoadMap:

PN is attempting to keep to a minimum of one release every two months. 
The planned feature-implementation schedule can be seen below. This will
be modified as and when I see fit - so don't bet money on it or anything.

last updated: 03/12/2003

0.5 Mingus
==========

Start help file.
Pre-defined tools support.
[Implemented] Menu to choose file encoding type (currently PN just preserves the original type).
[Implemented] Header switch as a generic feature.
[Implemented] Basic Project Management - file grouping etc.
[Implemented] Better printing - margins, header and footer.
[Implemented] Combo-box toolbar for scheme selection.
[Implemented - Bjoern Graf] Find & Replace actual combo-box entries (not just auto-complete).
[Implemented - Bjoern Graf] HTML Export.

0.6 Ella
========

Tab order management.
Method Locator - see http://www.pnotepad.org/articles/pn2methodlist.html
Basic Build Management - see http://www.pnotepad.org/articles/pn2projects.html
PCRE Based Search.
Filetype filters - e.g. allow loading of .dfm files.
HTML Export configuration, inline HTML export.

0.7 Basie
=========

Unicode build.
File browser window.
HTML preview window (ie+moz).

0.8 Ellington
=============

Mini-PN - true notepad-style document-centric PN alternative build.
